# Características

- A empresa foca apenas em fabricação de onsoles, deixando a distribuição e venda para terceiros
- Os produtos são vendido globalmente

# Objetivos

- [ X ] Consolidar todas as bases de terceiros para realizar uma análise
- [ X ] Transformar dados de vendas em informações relevantes para a fabricante
- [ X ] Quais são os produtos mais populares em cada país
- [ X ] Como otimizar o processo de transporte e logística até o momento da venda


No arquivo "chatgpt_prompts.md" inclui os prompts utilizados (LECGS) e as respostas do ChatGPT, foram 5 interações para que tenhamos uma planilha com os dados consolidados, separados por visão, com gráficos em imagens, poderia melhorar para gráficos dinâmicos do Excel, e ainda uma apresentção executiva para mostrar os dados para parceiros.

Analises realizadas
As 5 análises estratégicas recomendadas foram executadas:

1. Performance por Produto
    Ranking de produtos;
    Produtos mais vendidos;
    Produtos com maior receita;
    Identificação de concentração de faturamento.
2. Performance por Marketplace
    Comparativo de receita;
    Volume de pedidos;
    Ticket médio;
    Impacto dos descontos.
3. Análise Temporal
    Evolução mensal;
    Tendência de crescimento;
    Identificação de sazonalidade.
4. Perfil de Mercado / Países
    Países com maior demanda;
    Mercados prioritários;
    Oportunidades internacionais.
5. Indicadores Executivos
    GMV total;
    Ticket médio geral;
    Volume total vendido;
    Receita consolidada;
    Indicadores comparativos.


Principais insights logísticos adicionados:

    criação de hubs regionais;
    posicionamento estratégico de estoque;
    otimização de fulfillment;
    consolidação de fretes internacionais;
    previsão de demanda;
    automação logística;
    otimização de marketplaces por país;
    redução de custo operacional;
    melhoria de SLA de entrega;
    aumento de eficiência de transporte.
